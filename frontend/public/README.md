# Logo

Placez **`coly.png`** (votre logo) dans ce dossier :  
`frontend/public/coly.png`

Utilisation :
- **Page de connexion** : logo au-dessus du titre « Plateforme SST »
- **Barre latérale** : logo à côté du titre (clic → tableau de bord)
- **Barre supérieure** : petit logo cliquable à gauche du titre de page
- **Favicon** : l’onglet du navigateur utilise ce logo

Si le fichier est absent, l’icône SST et le message « Placez coly.png dans frontend/public/ » s’affichent.
